# 📘 Assignment: Statistics with Python

## 🎯 Objective

Learn to perform data analysis and basic statistical calculations using Python libraries pandas and numpy. You will analyze a dataset, compute summary statistics, and visualize results.

## 📝 Tasks

### 🛠️ Data Analysis with Pandas

#### Description
Load a CSV dataset and use pandas to explore and summarize the data.

#### Requirements
Completed program should:

- Load a CSV file into a pandas DataFrame
- Display the first 5 rows of the dataset
- Show summary statistics (mean, median, mode, std) for all numeric columns


### 🛠️ Statistical Calculations with Numpy

#### Description
Use numpy to perform additional statistical calculations on the dataset.

#### Requirements
Completed program should:

- Calculate the correlation matrix for numeric columns
- Identify columns with the highest correlation
- Compute and print the variance for each numeric column
